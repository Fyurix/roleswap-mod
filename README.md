# RoleSwap — челлендж "слепой / немой / глухой" в Minecraft

Fabric-мод: каждые N минут (по умолчанию 20) три игрока меняются ролями по кругу:

- **BLIND** — эффект слепоты (ванильный Minecraft-эффект).
- **MUTE** — не может говорить: сервер отменяет пакеты с его микрофона в Simple Voice Chat, остальных слышит нормально.
- **DEAF** — не слышит остальных: сервер отменяет звук, который летит именно к нему, сам говорить может.

## Что нужно перед сборкой

1. **На сервере и у всех клиентов должен быть установлен полноценный Simple Voice Chat** (не только API) — этот мод только цепляется к нему, сам голосовой чат не реализует. Скачать: modrinth.com/plugin/simple-voice-chat.
2. Открой `gradle.properties` и поставь версии `minecraft_version` / `voicechat_api_version`, совпадающие с версией Simple Voice Chat на твоём сервере (посмотри на Modrinth, какая версия API соответствует твоей версии мода).
3. Установлен JDK 21 и Gradle (или используй `gradlew`, если добавишь wrapper: `gradle wrapper`).

## Сборка

```bash
./gradlew build
```

Готовый jar будет в `build/libs/`. Кидаешь его в папку `mods` на сервере (и, если хочешь клиентские фичи — на клиенты, но для этой логики хватает серверной стороны).

## Команды

- `/roleswap start <игрок1> <игрок2> <игрок3>` — назначает роли по кругу (1-й слепой, 2-й немой, 3-й глухой) и запускает таймер.
- `/roleswap interval <минуты>` — меняет интервал ротации на лету (можно вызвать до или во время игры).
- `/roleswap stop` — останавливает челлендж и снимает эффекты.

## Известные места, которые стоит проверить перед использованием

Я собрал код по официальной javadoc Simple Voice Chat API (`voicechat.modrepo.de`), но у меня нет возможности прямо сейчас скомпилировать проект и проверить его на реальном сервере. Особое внимание удели:

1. **`VoicechatConnection.getPlayer().getUuid()`** в `VoicechatIntegration.java` — точное название метода получения UUID игрока может немного отличаться между версиями API. Если Gradle ругнётся на компиляции — открой javadoc твоей версии (`https://voicechat.modrepo.de/`) и поправь по автодополнению в IDE.
2. **Версии в `gradle.properties`** — `minecraft_version`, `yarn_mappings`, `loader_version`, `fabric_version`, `voicechat_api_version` должны быть согласованы между собой. Проще всего сгенерировать стартовый шаблон через https://fabricmc.net/develop/template/ под нужную версию Minecraft и перенести туда файлы из `src/main/java`.
3. Реальный **entrypoint id `"voicechat"`** в `fabric.mod.json` — в некоторых версиях API он может называться иначе, сверься с примерами аддонов на GitHub (`henkelmax/simple-voice-chat`, раздел addons/примеры).

## Идея для видео

Раз это для контента — стоит повесить на экран (через action bar или boss bar) индикатор "кто сейчас кто", чтобы зрителям было видно роли в реальном времени. Могу дописать это отдельным модулем, если нужно.
